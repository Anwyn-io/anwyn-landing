# Anwyn | The Privacy Sanctuary

**Anwyn** is a privacy-first "Walled Garden" for your personal data built on the **Midnight Network**. We provide a secure environment for users to safeguard their digital identity while participating in anonymous surveys.

## 🌿 The Vision
In the current market data landscape, users often trade personal privacy for rewards. Anwyn breaks this trade-off by utilizing Zero-Knowledge (ZK) technology to create a "Shadow Vault" for sensitive data. This preserves privacy and keeps your raw data secure while allowing authorized parties to query this data and get high-quality, verifiable information.

## 🛡️ Key Features
- **Data Cloaking:** Protects user identity, location, and metadata to prevent a hackable 'honeypot'.
- **Sovereign Yield:** Enables secure accumulation of USDCx rewards in return for providing valuable market data.
- **Gamified Token Mechanics:** Provides strong incentives for networking and community growth.

## 🏗️ Tech Stack
- **Blockchain:** Midnight Network (Cardano Sidechain)
- **Infrastructure:** Website and Mobile application
- **Frontend:** HTML/CSS (Hosted via Vercel)
